audio

  • src
    • sumber file
  • controls
    • agar user bisa mengendalikan audio-nya menggunakan controller default, jika tidak dipakai asumsinya akan dijalankan menggunakan javascript
  • preload
    • membuat file diload dulu sebelum dijalankan
  • autoplay
    • untuk menjalankan audio secara otomatis
  • loop
    • mengulang audio jika sudah selesai
  • muted
    • agar audio tidak ada suaranya
  • format file: .mp3, .wav, .ogg
